# 🗺️ U.S. States Game

A fun and interactive geographical quiz built using Python's `turtle` graphics module and `pandas`. In this game, your goal is to name all 50 U.S. states, one by one. Each correct guess displays the state's name on a blank map of the United States.

---

## 📌 Features

- 🐢 **Turtle Graphics Map Display**  
  A blank U.S. map is shown using the turtle module, and correct guesses are written on the map at their correct locations.

- 🎯 **Live Scoreboard**  
  A live score is displayed at the top of the screen, updating with each correct guess.

- 📋 **State Validation**  
  Input is validated using a list of 50 U.S. states loaded from a CSV file. Only unique and correct guesses are accepted.

- 🛑 **Exit Option**  
  Type `"exit"` at any point to quit the game early.
